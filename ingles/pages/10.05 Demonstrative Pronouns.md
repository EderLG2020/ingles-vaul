# Demonstrative Pronouns

## Objetivo

Aprender a usar:

- this
- that
- these
- those

para señalar personas, objetos, animales o situaciones según:

- distancia
- número (singular o plural)

---

# Definición

Los **Demonstrative Pronouns** y **Demonstrative Adjectives** se utilizan para señalar algo específico.

Permiten indicar:

- si algo está cerca o lejos
- si es singular o plural

---

# Tabla principal

| Distancia | Singular | Plural |
|---|---|---|
| Cerca | this | these |
| Lejos | that | those |

---

# Traducción general

| Palabra | Traducción posible |
|---|---|
| this | este / esta / esto |
| these | estos / estas |
| that | ese / esa / eso |
| those | esos / esas |

La traducción exacta depende del contexto.

---

| Word | Pronunciación aproximada |
|---|---|

| that | dat |
| those | dous |

> El sonido real utiliza el sonido inglés “TH”.

# Uso según distancia y cantidad

| Word | Distancia | Número |
|---|---|---|
| this | cerca | singular |
| these | cerca | plural |
| that | lejos | singular |
| those | lejos | plural |

---

# Estructuras básicas

## Singular

```txt
This/That + is + noun
```

### Ejemplos

- This is my phone.
- This is a good idea.
- That is my car.
- That is expensive.

---

## Plural

```txt
These/Those + are + noun
```

### Ejemplos

- These are my books.
- These are new.
- Those are my friends.
- Those are expensive.

---

# This

## Uso

“This” se utiliza para señalar algo singular que está cerca.

## Estructura

```txt
This + is + singular noun
```

---

## Ejemplos básicos

| Inglés | Español |
|---|---|
| This is my phone. | Este es mi teléfono. |
| This book is interesting. | Este libro es interesante. |
| This chair is comfortable. | Esta silla es cómoda. |
| This is easy. | Esto es fácil. |
| This computer is fast. | Esta computadora es rápida. |

---

## Ejemplos en contexto

| Situación | Ejemplo |
|---|---|
| mostrando un objeto cercano | This is my laptop. |
| presentando una idea | This is important. |
| señalando una persona cercana | This is my friend, Carlos. |

---

# These

## Uso

“These” se utiliza para señalar cosas o personas en plural que están cerca.

## Estructura

```txt
These + are + plural noun
```

---

## Ejemplos básicos

| Inglés | Español |
|---|---|
| These are my shoes. | Estos son mis zapatos. |
| These books are new. | Estos libros son nuevos. |
| These are my keys. | Estas son mis llaves. |
| These students are smart. | Estos estudiantes son inteligentes. |
| These cookies are delicious. | Estas galletas son deliciosas. |

---

## Ejemplos en contexto

| Situación | Ejemplo |
|---|---|
| mostrando varios objetos cercanos | These are my notebooks. |
| hablando de personas cercanas | These are my cousins. |
| señalando opciones | These are better. |

---

# That

## Uso

“That” se utiliza para señalar algo singular que está lejos.

## Estructura

```txt
That + is + singular noun
```

---

## Ejemplos básicos

| Inglés | Español |
|---|---|
| That is my car. | Ese es mi auto. |
| That house is beautiful. | Esa casa es hermosa. |
| That is expensive. | Eso es caro. |
| That movie was amazing. | Esa película fue increíble. |
| That restaurant is good. | Ese restaurante es bueno. |

---

## Ejemplos en contexto

| Situación | Ejemplo |
|---|---|
| señalando algo lejano | That building is very tall. |
| hablando de una idea pasada | That was a good decision. |
| señalando una persona lejana | That is my teacher. |

---

# Those

## Uso

“Those” se utiliza para señalar cosas o personas en plural que están lejos.

## Estructura

```txt
Those + are + plural noun
```

---

## Ejemplos básicos

| Inglés | Español |
|---|---|
| Those are my friends. | Esos son mis amigos. |
| Those cars are expensive. | Esos autos son caros. |
| Those houses are big. | Esas casas son grandes. |
| Those shoes look nice. | Esos zapatos se ven bien. |
| Those students are noisy. | Esos estudiantes son ruidosos. |

---

## Ejemplos en contexto

| Situación | Ejemplo |
|---|---|
| señalando objetos lejanos | Those are old buildings. |
| hablando de personas lejanas | Those are my neighbors. |
| comparando opciones | Those are better than these. |

# Demonstrative Pronouns vs Demonstrative Adjectives

## Como pronombre

Reemplaza el sustantivo.

| Ejemplo |
|---|
| This is mine. |
| Those are expensive. |

---

## Como adjetivo

Acompaña al sustantivo.

| Ejemplo |
|---|
| This phone is mine. |
| Those shoes are expensive. |

---

# Errores comunes

| Incorrecto | Correcto |
|---|---|
| This are my books. | This is my book. |
| These is my friend. | These are my friends. |
| That are expensive. | That is expensive. |
| Those is my shoes. | Those are my shoes. |

---

# Reglas importantes

## 1. This y That usan “is”

| Correcto |
|---|
| This is |
| That is |

---

## 2. These y Those usan “are”

| Correcto |
|---|
| These are |
| Those are |

---

## 3. This y That son singulares

| Correcto |
|---|
| this car |
| that house |

---

## 4. These y Those son plurales

| Correcto |
|---|
| these cars |
| those houses |
